# ReactJS and ReactContext Facebook Clone
Hosted in [Firebase](https://facebook-clone-c8e80.web.app)

Features:
- Google Auth Login
- Responsive homepage UI
- Functional NoSQL Database (FireStore) for seeing posts


To start:
```
npm start
```
To deploy:
In console project root:
```
npm run build
firebase deploy
```
